Classes and IDs in CSS

A div is a rectangle used to group different parts of a webpage, or "div-ide" it, so to speak:

the height, width, background color, text styles can all be assigned by giving a div a class.

Here are some examples...

I have put this text in a powder blue box by assigning <div> the class .powderblue

in between the <style> tags: .powderblue {background-color:#B0E0E6;
height:200px;
width:200px;
}



I have put this text in a snow colored box by assigning <div> the class .snow

in between the <style> tags: .snow {background-color:#FFFAFA;
height:200px;
width:200px;
}



I have put this text in a ivory colored box by assigning <div> the class .antique

in between the <style> tags: .antique {background-color:#FAEBD7;
height:200px;
width:200px;
}
