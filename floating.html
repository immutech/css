Classes and IDs in CSS

We can use the float value in the css code to make an area within a div tag behave

a certain way. Here are the boxes from the DIV lesson, but with float values added to the css.

Watch carefully...

.powderblue {background-color:#B0E0E6;
height:200px;
width:200px;
float:left;
}



.snow {background-color:#FFFAFA;
height:200px;
width:200px;
float:right;
}



.antique {background-color:#FAEBD7;
height:200px;
width:200px;
clear:both;
}

in the final box I added a "clear" value to make it appear under the boxes above.
