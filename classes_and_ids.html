Classes and IDs in CSS

I have made this text a steely color by assigning <p> the color:#B0C4DE (HEX CODE)



This text is 200% larger because in the html of the body, the p code has been

given an extra class like this: <p class="large" >

in between the <style> tags, the class is represented with a dot in front of the name, eg:

.large {font-size:200%;}



This text is 50% smaller because in the html of the body, the p code has been

given an extra class like this: <p class="small" >.

in between the <style> tags, the class is represented with a dot in front of the name, eg:

.small {font-size:50%;}



This like is in bold because in the body html I gave it an id of bold, like this:

<p id="bold" >

in between the <style> tags, the id is represented with a # (hashtag) in front of the name, eg:

#bold {font-weight:bold;}

IDs are usually assigned to single elements in a webpage, where as classes can apply to more than one element.



The word BUTTERFLY is underlined because in the body html I wrapped the word in <span> tags:

<span>BUTTERFLY</span>

in between the <style> tags, the span is just written as span, eg:

span{text-decoration:underline;}



Elements can be given more than one type of style value.
