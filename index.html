INTERNAL CSS

The reason this text is green is because in the <head> section of this site there is a

pair of <style> tags. the opening <style> tag has a type variable with a value

of text/css like this:

<style type="text/css"> .


after the <style> tag there is a p followed by {} (curly brackets).

This refers to the <p> in the html code. anything inside a <p> tag will be styled

according to the instructions in between the curly brackets.


In this case the instruction is color:green.
